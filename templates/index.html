Constraint Handling

✓
Same Start & Goal:

If you select the same location for both start and destination, the system will notify you that you're already at your destination.

⚠
Unknown Locations:

If you enter an invalid location, the system returns an error and shows a list of valid locations.

↔
Multiple Optimal Paths:

When multiple paths have the same distance, the system lists all options and displays them in different colors on the map.

●
●
●
Road Conditions:

Roads are marked as good, moderate, or poor. You'll receive warnings for poor road conditions.

🚗
Traffic Hotspots:

Major areas like Meskel Square, Megenagna, Bole Michael, and Piazza are marked as high traffic zones. You'll be warned when starting from these locations.
